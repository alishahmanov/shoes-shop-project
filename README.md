# Shoes Shop Project

Современный интернет-магазин обуви с умной фильтрацией, корзиной и адаптивным дизайном.

## Описание

Shoes Shop — полнофункциональный веб-сайт магазина обуви, разработанный на чистом HTML, CSS и JavaScript. Включает 10 страниц, систему фильтрации товаров, корзину покупок и темную тему.

## Основные возможности

### Каталог товаров

- 16 моделей обуви (Nike, Adidas, Puma, Jordan, New Balance, Reebok, Asics, Vans, DC, Crocs, Skechers, Under Armour)
- Фильтрация по бренду, категории, цвету и цене
- Кросс-страничный поиск — товары загружаются со всех страниц автоматически
- Умная пагинация — скрывается при активных фильтрах

### Корзина покупок

- Добавление товаров из любой страницы каталога
- Управление количеством (кнопки +/−)
- Автоматический расчет итоговой суммы
- Сохранение в localStorage — данные не теряются при перезагрузке
- Счетчик товаров в шапке сайта
- Toast-уведомления при добавлении

### Конструктор индивидуального заказа

- Выбор модели обуви (8 вариантов)
- Настройка размера и количества
- Выбор цвета с визуальным превью
- Выбор материала: кожа, замша, текстиль
- Выбор стельки: стандарт, ортопедическая, спортивная
- Дополнительные опции: светоотражатели, водозащита, вентиляция
- Способы доставки: самовывоз, курьер, почта, экспресс
- Загрузка референсного изображения
- Калькулятор с автоматическим расчетом цены
- Валидация формы

### Дополнительные функции

- **Темная тема** — переключение между светлым и темным режимами с сохранением выбора
- **Поиск товаров** — с автодополнением и мгновенной фильтрацией
- **Отзывы клиентов** — форма добавления отзывов с рейтингами (1-5 звезд)
- **FAQ** — поиск по вопросам и фильтрация по категориям
- **Адаптивный дизайн** — mobile-first подход, работает на всех устройствах
- **Живые часы** — дата и время в реальном времени на главной странице

## Технологии

- **HTML5** — семантическая разметка, ARIA-атрибуты для доступности
- **CSS3** — Flexbox, Grid, CSS Variables, медиа-запросы, анимации
- **JavaScript ES6+** — async/await, Fetch API, localStorage, модульная архитектура
- **Vanilla JavaScript** — без использования фреймворков (React, Vue и т.д.)

## Установка и запуск

### Требования

- Современный браузер (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Локальный веб-сервер (необходим для работы Fetch API)

### Быстрый старт

**1. Клонируйте репозиторий:**

```bash
git clone https://github.com/alishahmanov/shoes-shop-project.git
cd shoes-shop-project
```

**2. Запустите локальный сервер (выберите один вариант):**

Python 3:

```bash
python -m http.server 8000
```

Node.js:

```bash
npx http-server -p 8000
```

PHP:

```bash
php -S localhost:8000
```

VS Code Live Server:

- Установите расширение "Live Server"
- Откройте проект в VS Code
- Нажмите "Go Live" в статус-баре

**3. Откройте браузер:**

```
http://localhost:8000
```

## Использование

### Фильтрация товаров

1. Откройте страницу категорий
2. Выберите фильтры в левой панели (бренд, категория, цвет)
3. Выберите ценовой диапазон в шапке
4. Товары отфильтруются автоматически со всех страниц
5. Снимите фильтры для возврата к стандартному виду

### Работа с корзиной

1. Нажмите "В корзину" на любом товаре
2. Перейдите в корзину через шапку сайта
3. Используйте кнопки +/− для изменения количества
4. Нажмите "×" для удаления товара
5. Итоговая сумма пересчитывается автоматически

### Создание индивидуального заказа

1. Откройте "Форма под заказ"
2. Выберите модель, размер и количество
3. Настройте цвет, материал и стельку
4. Отметьте дополнительные опции
5. Выберите способ доставки
6. Заполните контактные данные
7. Цена рассчитывается автоматически

## Ключевые особенности

### Кросс-страничная фильтрация

Товары автоматически загружаются со всех страниц каталога через Fetch API. При активных фильтрах показываются все подходящие товары, пагинация скрывается.

### LocalStorage

- Корзина сохраняется автоматически
- Темная тема запоминается
- Данные не теряются при перезагрузке страницы

### Адаптивность

- Mobile-first подход
- Мобильные устройства (320px+)
- Планшеты (768px+)
- Десктоп (1024px+)
- Hamburger-меню на мобильных
- Touch-friendly элементы
